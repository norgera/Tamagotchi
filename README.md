### build:
```bash
javac -cp .:junit-jupiter-api-5.11.3.jar *.java
```
unit tests (optional):
```bash
java -cp .:junit-jupiter-api-5.11.3.jar org.junit.runner.JUnitCore <TestClassName>
```

Run:
```bash
java Main
```
